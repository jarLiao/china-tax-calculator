# 中国个税计算器 v1.1.0

专业、准确的中国个人所得税计算工具，符合2025年最新税法规定。

## 🎉 v1.1.0 新增功能

| 功能 | 说明 |
|------|------|
| 📊 **年度汇算清缴** | 计算年度应补/应退税额 |
| 💼 **多收入来源** | 工资+劳务+稿酬+特许权使用费 |
| 🎯 **年终奖智能优化** | 自动找到最优分配方案 |
| 💬 **跳槽薪资谈判** | 税后反推税前，生成谈判话术 |
| 📋 **Offer 对比** | 多个 Offer 一键对比 |

## 核心功能

| 功能 | 说明 |
|------|------|
| 💰 **月度工资个税** | 累计预扣法计算，支持多档税率 |
| 🧧 **年终奖计税** | 单独计税 / 合并计税对比 |
| 📋 **专项附加扣除** | 7项扣除自动计算 |
| 🔄 **税后反推税前** | 已知到手工资，反推税前 |
| 📊 **个税报告** | 生成详细的个税计算报告 |

## 使用示例

### 1. 计算月度工资个税

```
我月薪30000元，五险一金4500元，有1个孩子上学，房贷每月10000元，赡养老人（非独生子女，分摊1000元），帮我算一下个税和到手工资
```

### 2. 年终奖计税对比

```
年终奖60000元，月薪20000元，帮我对比单独计税和合并计税哪个更划算
```

### 3. 年终奖智能优化（新功能）

```
我有10万年终奖，月薪2万，怎么分配税最少？帮我找到最优方案
```

### 4. 年度汇算清缴（新功能）

```
我年薪36万，劳务报酬2万，稿酬1万，五险一金每月4500，有1个孩子上学和房贷，已预缴个税25000，帮我算一下年度汇算要补税还是退税
```

### 5. 跳槽薪资谈判（新功能）

```
我想每个月到手25000元，五险一金4000元，有子女教育和房贷扣除，税前工资要多少？帮我生成谈判话术
```

### 6. Offer 对比（新功能）

```
我有3个Offer：A公司月薪25k，年终奖5万；B公司月薪28k，年终奖3万；C公司月薪22k，年终奖8万。五险一金都是月薪的16%，帮我对比哪个最划算
```

### 7. 税后反推税前

```
我想每个月到手25000元，五险一金4000元，有子女教育和房贷扣除，税前工资要多少？
```

### 8. 多收入来源（新功能）

```
我工资年薪30万，还有劳务报酬3万，稿酬1万，怎么算个税？
```

## 注意事项

1. **累计预扣法**：每月个税按年初累计计算，非简单月度计算
2. **年终奖优惠**：2027年12月31日前，年终奖可选择单独计税
3. **专项扣除**：五险一金中的个人缴纳部分可税前扣除
4. **赡养老人**：仅限生父母、继父母、养父母，不包括岳父岳母/公婆

## 计算公式

### 月度个税（累计预扣法）

```
累计应纳税所得额 = 累计收入 - 累计免税收入 - 累计专项扣除 - 累计专项附加扣除 - 累计其他扣除 - 5000×累计月份

本期应纳税额 = (累计应纳税所得额 × 税率 - 速算扣除数) - 累计已预缴税额
```

### 年终奖单独计税

```
月应纳税所得额 = 年终奖 ÷ 12
对应税率和速算扣除数
应纳税额 = 年终奖 × 税率 - 速算扣除数
```
